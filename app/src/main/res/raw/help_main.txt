Explorers - Quickly view or calculate values without storing any data.
--> Transaction Explorer - Manually enter transactions to get net values of assets.
--> Address Explorer - View all balances and transactions on an address.
--> Exchange Explorer* - View all balances and transactions on an exchange account.
--> Chart Explorer - View charts of cryptos vs. fiats.

Portfolios - Store collections of data.
--> Transaction Portfolio - Store a collection of transactions.
--> Address Portfolio - Store a collection of addresses.
--> Exchange Portfolio* - Store a collection of exchange accounts.
--> Chart Portfolio - Store a collection of charts.

Crypto Tools:
--> Crypto Prices - View up-to-date prices for various crypto.
--> Crypto Converter - Convert crypto quantities.
--> Reflections Calculator* - Estimate the reflections earned for a crypto on an address.

Managers:
--> Data Management - Import* and export* data, delete data, and reset the app.
--> Fiat Manager - Add your own custom fiats.
--> Coin Manager - Add your own custom coins.
--> Token Manager* - Download token lists or add your own custom tokens.


*Premium Feature (Requires In-app Purchase)
